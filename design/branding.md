# Branding & Visual Language

Source: OSHIT UI Figma file.

---

## Colors

| Role | Value |
|------|-------|
| Primary | `#FFD73B` / `#FFCB00` |
| Hero background gradient | `linear-gradient(to top, white, #ffe78b)` |
| Accent (token amounts) | `#F47900` |
| Card border | `#fff6d5` / `#fffaea` |
| Action card gradient | `rgba(255,203,0,0.2) → rgba(193,255,0,0.2)` |
| Heading text | `#272727` |
| Secondary text | `#898989` / `#616161` |
| Brand display text | `#fffdf4` |
| Footer / caption text | `#858585` |
| Page background | `#ffffff` |

---

## Typography

**Primary font:** Nunito  
**Fallback:** Noto Sans JP (CJK), sans-serif

| Use | Weight | Size |
|-----|--------|------|
| Brand display (e.g. "OSHIT") | Black 900 | 44px |
| Section heading | Black 900 | 20px |
| Card title | Bold 700 | 12–14px |
| Body / description | Medium 500 | 12–14px |
| Caption / meta | Regular 400 | 11–12px |

---

## Spacing & Layout

- Base width: 375px (mobile-first)
- Horizontal padding: 20–26px
- Card border radius: 10–12px
- Card grid: 2-column

---

## Asset Fallback Rule

If a required image or icon asset cannot be found in the project's asset folders, use an appropriate emoji as fallback. Do not leave broken image references or empty spaces.
